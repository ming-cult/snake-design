## 介绍

基于 `React hooks` 开发的 `PC` 端组件库。[文档地址](https://ming-cult.github.io/snake-design)

### 特性

- 组件均基于 hooks 开发
- 除了特殊说明都为受控组件
- 基于 TDD 开发, 90% 以上的测试覆盖率

> [组件开发进度](https://github.com/ming-cult/snake-design/projects/1)

## 命令说明

- `yarn start`: 启动项目
- `yarn test`: 运行单元测试
- `yarn test:coverage`: 查看代码覆盖率
- `yarn build`: 编译打包组件
- `yarn publish`: 自动化发布版本
